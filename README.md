# Django-Shepherd.github.io

Personal homepage for GitHub Pages.

## Publish

1. Create a new public GitHub repository named `Django-Shepherd.github.io`.
2. Push this folder to that repository.
3. Open `https://django-shepherd.github.io` after GitHub Pages finishes deploying.

## Edit

- Update the name, biography, and links in `index.html`.
- Replace `assets/hero.jpg` with another image if you want a different visual style.
- Change colors and layout in `styles.css`.
